Write a function called `addValue` which takes two parameters:

- `obj` -  an object 
- `key` - a string

The function should add a new key-value pair to the given object. Set the key's value to `true`.

**For example:**

```
var myObj = {};
```

```
addValue(myObj, 'myValue');
```

```
console.log(myObj['myValue']); // should print true
```
